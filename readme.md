# Pokémon Sprites Gallery

This project displays Pokémon sprites, including both normal and shiny variants, organized for easy viewing and development purposes.

## Project Structure

- `index.html` — Main webpage displaying Pokémon images.
- `style.css` — Styles for the webpage.
- `script.js` — JavaScript for interactivity (if needed).
- `assets/` — Folder containing sprite images.
  - `sprites/` — Normal sprites.
  - `shiny_sprites/` — Shiny sprites.

## How to Use

- Open `index.html` in your browser to view the Pokémon sprites.
- Images are organized in separate folders for normal and shiny sprites.
- You can add or modify sprites by updating the images in the `assets/` folders.

## Sources

All Pokémon sprites are sourced from [Dragonfly Cave Resources](https://www.dragonflycave.com/resources/sprites/). These sprites are used for educational and personal project purposes.
